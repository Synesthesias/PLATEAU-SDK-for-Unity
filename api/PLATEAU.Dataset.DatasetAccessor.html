    Class DatasetAccessor

    GMLファイル群から利用可能なファイル、メッシュコード、LODを検索します。 C++の内部ではこれは基底クラスとなっており、継承によりローカル向けとサーバー向けの両方に対応しています。 このクラスのポインタ (Handle) の具体的な型がローカル向けとサーバー向けのどちらであるかは、 DatasetSource の初期化時に指定し、 Accessor でその型である DatasetAccessor を取得します。
