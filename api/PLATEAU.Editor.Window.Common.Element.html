    Class Element

    GUIの1要素を抽象化したクラスです。 描画のON/OFFを切り替え機能や、 PLATEAU.Editor.Window.Common.ElementGroupから検索されるための名前を有します。 このクラスを継承してボタンやオブジェクトフィールドなどGUI要素を構築できます。 または、GeneralElementを利用して任意の描画メソッドを実行できます。
