    DllLogger

    DLL側のログをC#側にコールバックで転送するクラスです。 C#側でログを表示するためのコールバックメソッドとログ詳細度を指定できます。

    DLL側のログをC#側でコールバックで受け取ることができますが、 その Error, Warn, Info の3つのコールバックをまとめたクラスです。 詳しくは DllLogger をご覧ください。

    PInvokeDisposable

    廃棄時に Native側で delete する必要があるものの基底クラスです。 Native側で delete する処理として、 DisposeNative() がサブクラスで実装されていることを前提とします。 廃棄タイミングは GC処理時 または using(var a){} ブロックを抜ける時 または Dispose() を呼んだときです。
