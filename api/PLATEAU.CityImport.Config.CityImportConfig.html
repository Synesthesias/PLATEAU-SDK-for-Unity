    Class CityImportConfig

    都市インポートの設定です。 インポート画面の設定GUIでユーザーはこの設定値を書き換えていくことになります。 パッケージごとのインポート設定についてはPackageImportConfigDictを参照してください。 設定GUIについてはCityAddGUIおよびCityLoadConfigGUIを参照してください。
